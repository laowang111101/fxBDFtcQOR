# 购物商城管理系统 python208

## 项目概述

本项目是一个基于Python开发的购物商城管理系统，采用PyQt5进行界面设计，MySql数据库进行数据存储。

## 技术栈

- **界面设计：** PyQt5
- **数据库：** MySql 5.7+

## 功能模块

### 商家功能

- 注册与登录
- 修改店铺信息
- 商品管理：添加、删除、修改、查找商品
- 查看交易记录

### 顾客功能

- 注册与登录
- 修改收货信息
- 商品查找与购买
- 查看交易记录
- 取消订单

## 系统角色

系统包含两个主要角色：商家和顾客。

## 运行环境

- Python环境
- MySql数据库环境

## 目录结构

```plaintext
购物商城管理系统/
├── data/             # 数据库文件目录
├── src/              # 源代码目录
│   ├── main.py       # 程序主入口
│   ├── merchant.py   # 商家功能模块
│   └── customer.py   # 顾客功能模块
└── UI/               # 界面文件目录
    ├── main_window.ui
    └── ...
```

## 部署步骤

1. 确保Python和MySql环境正确安装。
2. 导入数据库文件至MySql。
3. 运行main.py启动程序。

## 核心亮点

- **清晰的业务逻辑：** 功能模块明确，易于理解和扩展。
- **友好的用户界面：** 基于PyQt5的图形用户界面，操作简便。
- **灵活的数据库设计：** 使用MySql进行数据存储，支持数据的高效管理。
